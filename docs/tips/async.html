    Asynchronious wrapped operations

    Warning

    Only .Net Core Standard 2.0 version supported asynchronious wrapped operations

    To access data with XData inside async methods it's handy to use asynchronous wrappers declared inside XData.Async etxtension methods as illustrated in followed example:

    using XData.Async;
...
internal async Task<DocType[]> TestArrayAsync(IDataEngine dataEngine, CancellationToken cancellationToken)
{
    using (var dataScope = dataEngine.NewDataScope("TEST_POSTGRE_NPG"))
    {
        return await dataScope.GetRepository<DocType>().ToArrayAsync(cancellationToken);
    }
}

    Using asynchronously diposable objects requires an additional package XData net standard 2.0 Asynchronous Disposable support package and supporting of .Net Standard 2.1 specification

    To get asynchronous diposable version of...

    • IDataScope use NewDataScopeAsync or GetDataScopeAsync extension methods of IDataEngine.
    • ITransaction use BeginTransactionAsync extension methods of IDataScope.
    • IRepository<T> use GetRepositoryAsync<T> extension methods of IDataScope.
    using XData.Async;
using XData.AsyncDisposable;
...
internal async Task<DocType[]> TestArrayAsync(IDataEngine dataEngine, CancellationToken cancellationToken)
{
    await using (var dataScope = dataEngine.NewDataScopeAsync("TEST_POSTGRE_NPG"))
    {
        return await dataScope.GetRepository<DocType>().ToArrayAsync(cancellationToken);
    }
}
