Detailed Description

Data object filter combination (filter group) attribute

Filter combinations combine filters into named logical groups united by same logical operation (see XDataObject.Combination)

Default filter combination always exists. It have XDataObject.Combination.And combination and named as Empty string

Filter combination tree based on FilterCombinationAttribute.Parent property
